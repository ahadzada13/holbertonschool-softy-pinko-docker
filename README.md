# holbertonschool-softy-pinko-docker
Task 0: Set up the environment and get Docker ready.
Task 1: Built a Docker image for the Flask API (Back-end).
Task 2: Built a Docker image for the Nginx server (Front-end).
Task 3: Connected the front-end to the back-end using CORS.
Task 4: Used Docker Compose to manage both containers with one command.
Task 5: Added an Nginx Proxy to handle all requests through a single port 80.
Task 6: Scaled the API to multiple instances for Load Balancing.

How to Run
docker-compose up --build
